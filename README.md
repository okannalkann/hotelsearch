# Hotels

| Route | HTTP Verb	 | Description	 |
| --- | --- | --- |
| /hotels | `GET` | List all hotels. |
| /hotels/:star | `GET` | Hotels matching the given star. |


1) git clone https://github.com/okannalkann/hotelsearch.git .

2) npm i

3) npm start

4) Open browser

5) localhost:3000/hotels
